# Isolation Forest Based Anomaly Detection System

## Overview
This project implements an unsupervised anomaly detection system for network intrusion detection using the Isolation Forest algorithm. The system is designed to identify abnormal network behavior, including unknown and zero-day attacks, without relying on labeled data.
 

## Approach
- Data preprocessing (handling missing values and duplicates)  
- Feature scaling  
- Training an Isolation Forest model on normal data  
- Detecting anomalies based on deviation from learned patterns  

## System Workflow
1. Input network traffic dataset  
2. Data cleaning and preprocessing  
3. Feature scaling  
4. Model training  
5. Anomaly detection  
6. Evaluation  

## Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- F1-score  
- False Positive Rate  

## Technologies Used
- Python  
- Scikit-learn  
- Pandas  
- NumPy  

## Conclusion
The system provides an efficient and scalable solution for detecting anomalous network behavior and identifying unknown cyber threats using an unsupervised learning approach.
